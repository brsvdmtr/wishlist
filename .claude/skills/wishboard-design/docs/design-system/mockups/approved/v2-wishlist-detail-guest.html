North Star — Wishlist · guest

Момент дарения

Сюда приходят по share-ссылке. Пользователь 90% времени — дарильщик, не владелец. Экран должен снимать тревогу: «что дарить и как не ошибиться».

Owner card (верх экрана)

  • Avatar + имя + дата/count — реляционный якорь (кому ты даришь)
  • «+ Подписаться» CTA — вход в long-term отношения (PRO)
  • Тап по карточке → public-profile / showcase

Don't-gift preferences — встроенный блок

  • 🚫 иконка + red-soft background — визуально отделено от желаний
  • Preset-теги + custom note — не просто «аллергия», а контекст
  • В current prod это часто теряется; здесь — ПЕРВОЕ что видишь после owner'а

Item card × 5 states (guest view)

Available — обычный card, две CTA: ✓ Забронировать (primary accent) + 🔒 Тайно (secret-soft)
Reserved by me (public) — green-tint background, «✓ вы забронировали» chip + TTL countdown
Reserved by someone else — opacity 0.55, «⏳ уже забронировано» (имя НЕ раскрывается)
Reserved by me — secret (PRO) — accent-tint background, 🔒 chip, «👁‍🗨 владелец не знает»
Purchased — strike-through title, «💳 куплено» chip, opacity 0.45

Group-gift prompt после списка

Banner в promo-gradient — «собери 5 человек на одно желание». Не навязчиво, ниже всех items, узнаваемо-premium.

Что лучше current prod

  • Владелец — живой человек (avatar + подписка), не просто имя
  • Don't-gift выход на первый план — не после скролла
  • Reserve + Secret как 2 CTA рядом (а не secret в overflow)
  • State-hierarchy через background-tint (не только badge)
  • «Уже забронировано» без раскрытия имени (privacy по-умолчанию)
