Что отрисовано

Каждый «телефон» = одно состояние нового экрана search. Шрифты/цвета — исключительно токены v2.1 (--wb-card, --wb-accent-soft, --wb-r-xxl, 650 вес, mesh backdrop). Иконки — emoji + Unicode-glyphs, как требует iconography.

  • P1. Главный экран — где живёт кнопка 🔍 (контекст входа).
  • P2. First open — пусто, recent + quick filters.
  • P3. Short query (1 символ) — подсказка «введи ещё букву».
  • P4. Loading — skeleton-карточки.
  • P5. Full results (Free) — grouped + PRO-paywall блок.
  • P6. Filtered: только «Желания» + smart-chips.
  • P7. Empty — ничего не нашли.
  • P8. Restricted result — чужой вишлист недоступен.
  • P9. Error — поиск временно не сработал.
  • P10. PRO user — открыты брони, события, anti-gift.
  • P11. RTL (ar) — для проверки arabic.
