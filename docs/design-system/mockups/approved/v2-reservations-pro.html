North Star — Reservations PRO

Gift-giver workflow — уникальный surface WishBoard

Это не тот же самый экран что Reservations tab на Home. Home tab — быстрый обзор. Здесь — полный workflow управления бронями с PRO-фичами.

Phone 1 — Active с реальным PRO-surface

  • Summary tile×3: активных / тайных / в истории — sub-tabs навигация
  • Filter row: 6 фильтров (все / не куплено / куплено / с комментариями / тайные / группой) — выглядит длинно, но это реальные фильтры
  • Reservation card — tint меняется по modifier:
    • Smart active → green-border + TTL прогресс-bar
    • Secret → accent-tint + 🔒 chip
    • Group → rose-pink + «3/5 скинулись»
    • Expiring → amber + «⚠️ через 6ч» + «+ Продлить»
    • Done → strike + opacity 0.65 + «💳 куплено»
  • Note preview inline: «📝 Белый, с USB-C чехлом...» — italic, accent-tint bg
  • Reminder chip: «🔔 напомнит за сутки» — показывает что бот следит

Phone 2 — История + detail-sheet

  • History: compact rows сгруппированы по месяцам. «Апрель 2026», «Март 2026» — chronology
  • End-reason chips: ✓ куплено / ⏱ TTL истёк / ↶ отменено — цветовая маркировка
  • Detail sheet (bottom): нажатие на карточку открывает полный edit:
    • Item preview
    • TTL control с «+ Продлить на 48ч» + «осталось 2 из 3 продлений»
    • Note editor (inline, не отдельный экран)
    • «Куплено» toggle с success-check
    • Footer: «Отменить бронь» (ghost) / «Закрыть» (primary)

Почему это отдельный North Star surface

Обычные task-managers показывают список и всё. WishBoard конкурирует за gift-giver experience — это уникальная category. Здесь пересекаются 6 PRO-фич (TTL + note + reminder + purchase + filter + history) — как они работают вместе определяет value proposition.

Что лучше current prod

  • Detail-sheet объединяет все действия (TTL / note / mark purchased / cancel) в одно место — сейчас это разные экраны
  • Note-preview inline в карточке — сейчас скрыт до открытия
  • Extension limit «2 из 3» видимый — сейчас абстрактный
  • Reminder chip сообщает что бот следит — сейчас silent
  • End-reason в истории явный (TTL-expired vs cancelled vs purchased) — сейчас всё «завершено»
