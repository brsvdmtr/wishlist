Подарки для мамы
Отключить ссылку?
Ссылка станет недействительной без возможности восстановления.

Все, кому ты её отправлял, больше не смогут открыть её.

Если позже захочешь снова поделиться, нужно будет создать новую ссылку.
Вариант для профиля
‹ Управление ссылками
Закрыть профиль?
Профиль станет недоступен по ссылке.

Новые пользователи не смогут найти тебя, но текущие подписчики сохранят доступ к твоим вишлистам.

Ты сможешь открыть профиль снова в настройках приватности.
Пустое состояние
Когда у пользователя нет ни одной активной ссылки. Без CTA-кнопки, мягкое пояснение.

У тебя пока нет активных ссылок
Здесь будут ссылки на профиль, вишлисты и временные lite-вишлисты, которыми ты делишься с другими.
Поделись профилем или вишлистом, чтобы они появились здесь.
Экран недействительной ссылки
Показывается при переходе по отключённой или истёкшей ссылке. Не молчаливый redirect, а объяснение. CTA зависит от наличия вишлистов.
