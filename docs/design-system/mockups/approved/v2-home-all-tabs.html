North Star — Home × 3 tabs

Что покрывает эта сетка

Три реальных таба с кодового type HomeTab = 'wishlists' | 'wishes' | 'reservations'. Каждый таб — отдельный mental mode, не разные слайсы одной сущности.

Общий слой (все 3 таба)

  • Header: аватар (тап → профиль, где настройки) + greeting с локальным контекстом + правый slot (контекстно: PRO+⚙ / ⋮ / 🔍)
  • ⚙ Settings — отдельная иконка, явно видна в header'е на Wishlists-табе. Альтернативно настройки реально доступны и через тап по аватару (профиль) — iOS/Instagram pattern.
  • Tab-bar: pill с accent-fill на active, counter-badge как красный кружок top-right — нотификационная метафора, не inline-chip
  • Sticky CTA: отражает контекст — «создать вишлист» / «добавить желание» / отсутствует (брони не создаёшь отсюда)

Wishlists tab (первый)

  • Current wishlist — accent-glow + синий dot справа вверху («это активный»)
  • Shared wishlist — overlap-аватарки (реальная реляционная ценность)
  • Limit indicator — inline locked-tile «3/3 на FREE · Unlock» — не agressive paywall, а мягкий context-ual upsell
  • Подписки — отдельная секция, с PRO-бейджами у showcase-профилей

Wishes tab (flat cross-wishlist — ВОТ ЧТО НЕ БЫЛО В V1)

  • Header переключается на «все мои желания · 25 штук» — интент глобального обзора
  • Sticky quick-filters: priority (🙂😊😍 с реальными counts), статус, бюджет
  • Группировка: «по приоритету» / «забронированы» / «остальное» — не flat, но с логичной иерархией
  • Compact row: thumb + title + priority chip + price + source — всё в одну линию
  • Shared-wish indicator: «🔗 в 2» на карточке

Reservations tab (gift-giver workflow)

  • Header: «5 активных · 2 секретных 🔒»
  • Quick-filters: активные / куплено / тайные / группой / история
  • Smart reservation: TTL progress bar, countdown chip «через 5 дней»
  • Secret (PRO): карточка с accent-gradient, 🔒-индикатор, «владелец не знает»
  • Group gift: «👥 Скидываемся 3/5», роль «организатор»
  • Expiring warning: амбер border + «⚠️ истекает через 6ч» + красно-оранжевый TTL bar
  • Purchased: strike-through, opacity 0.7, 💳 chip, 3 890 ₽ как чек
  • История — locked-tile для FREE пользователей

Что лучше current prod

  • Все 3 таба — единый visual language. В current prod каждый таб слегка дрейфует.
  • State-chips унифицированы и читаются с одной секунды
  • Реальные upsell-tile вписаны в контент, не отдельной модалкой
  • Secret reservation визуально отличается от public (accent-gradient vs. success-soft)
  • Cross-wishlist Wishes-таб получает настоящую группировку с priority-секциями
