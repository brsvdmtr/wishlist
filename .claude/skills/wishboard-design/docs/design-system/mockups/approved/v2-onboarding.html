North Star — Onboarding

Первая встреча — эмоция, не форма

Onboarding — самый важный surface для retention. 4 экрана, каждый имеет чёткую роль. Warm, decisive, calm — все 3 principles сразу.

Screen 1 — Hello (эмоциональный якорь)

  • Floating gift emoji 120px с brand-glow radial, animation float — чувственный welcome, не logo
  • ❄ ✨ снежинки/искры по углам — лёгкий seasonal decor
  • Title: «Привет! Это WishBoard» — 30/800, accent-color на бренде
  • Sub: «говоришь о подарках... помогаешь не ошибиться» — объясняет dual-value: обе стороны
  • CTA: «Дальше» + ghost «У меня уже есть аккаунт»

Screen 2 — Why (value proposition)

  • 3 value-card с разными accent-tints (purple / green / amber) — emotional range
  • Пойми что хочешь · Поделись — получи подарки · Сюрприз не испорчен — 3 benefits с разговорными заголовками (не «feature list»)
  • Progress dots: done / active / pending / pending — пользователь видит где в процессе

Screen 3 — First wishlist setup (practical)

  • Question: «К какому событию ты собираешь?» — не «Create wishlist» form. Разговор, не action.
  • 6 occasions grid 2×3: 🎂 ДР · 🎄 НГ · 💍 Свадьба · 🎓 Выпуск · 🏡 Новоселье · 🎁 Просто так
  • Selected-state: accent-border + accent-soft bg + 2px ring-shadow
  • Click pattern: сразу автоматически создаёт wishlist и переходит в title-input (не сделан здесь)

Screen 4 — Success (celebration)

  • Success check 100px green gradient circle с pop-animation + box-shadow ring. Не плоский ✓.
  • Sparkle animation вокруг — легкий праздник, не cliché confetti
  • Title: «Готово 🎉» — эмодзи рядом
  • Tip-card: green/accent gradient, «💡 Совет» — добавь 3-5 желаний, поделись прямо в Telegram
  • 2 CTA: primary «+ Добавить желание» / ghost «Сразу поделиться» — обе валидны

Общие принципы

  • Dots pagination поверх sticky footer — compact indicator прогресса
  • Skip button (экраны 2-3) в правом углу — always-escape для power users
  • Primary-gradient CTA на каждом экране — один main action
  • Tone: разговорный русский. «пойми что хочешь», «поделись — получи». Не перевод с английского.

Что лучше current prod

  • Hello-экран с floating-gift animation — эмоциональный якорь, не logo-splash
  • Value prop переформулирован в 3 разговорные строки, не list of features
  • First-wishlist — question «к какому событию» вместо pure form
  • Success-экран с pop-animation + tip-card — celebrate a win
  • Tone русский-разговорный, не перевод
  • Progress-dots + skip — контроль для пользователя
