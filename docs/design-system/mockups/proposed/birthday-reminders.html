Примечания к макетам

Статус: DRAFT — ожидает ревью и аппрува


UI-примитивы:

  • Settings секции: SettingsSection, SettingsRow, SettingsToggle (Wave 4, provisional)
  • Баннеры: Banner с тонами accent / warning / info
  • Sheet: Sheet (canonical) для audience picker / custom message
  • Birthday badge: нужен новый Chip вариант warning или кастомный бейдж
  • Opt-in промпт: стандартный Sheet с карточкой

Токены:

  • Birthday accent: --wb-warning (#FBBF24) + --wb-warning-soft — уже есть
  • Birthday gradient: rgba(251,191,36,0.08) → rgba(251,191,36,0.02)
  • Не нужны новые токены — всё покрывается существующей warning palette

RTL (Arabic):

  • Все row-элементы зеркалятся через dir="rtl"
  • Chevron поворачивается: › → ‹
  • Toggle остаётся на месте (нативное поведение)
  • Текст выравнивается вправо

Не показано в макетах (реализуется стандартно):

  • Pro paywall context для birthday — аналогичен существующему calendar paywall
  • Загрузка / skeleton состояния
  • Анимация появления баннера
  • Pressed / disabled состояния кнопок
  • God Mode dashboard — серверный рендер, без UI
