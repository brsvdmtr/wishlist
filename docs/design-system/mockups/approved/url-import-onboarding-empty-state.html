«Добавить товар по ссылке» — онбординг в пустом состоянии

После переименования фичи (Импорт по ссылке → Добавить товар по ссылке) карточка стала понятнее, но экран, куда она ведёт — «Неразобранное» — по-прежнему встречает новичка сухим empty-state «Пусто». Что делать с полем ввода — не сказано.

Предложение: empty-state становится коротким онбордингом из 3 шагов — скопируй ссылку → вставь выше → дальше соберу сам. Показывается только при 0 черновиков; первый добавленный товар вытесняет онбординг списком (Frame 2). Состояние выводится из draftsItems.length — без localStorage и без отдельного экрана. Токены v2.1, inline CSS.
