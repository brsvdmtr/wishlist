Навигация и back-stack
Из детали назад: Back → «Мои брони» (одна страница, не двойной back).
После создания тайной: toast «🔒 Забронировано тайно» с action «Открыть» → сразу detail этой брони. Back → «Мои брони».
Из Settings: пункт «Мои тайные брони» ведёт в my-reservations с авто-скроллом к секции «🔒 Тайные» (или просто без скролла — уже сверху).
Unseen-dot на шестерёнке: гасится когда пользователь открыл «Мои брони» и увидел секцию «Тайные» (аналогично Santa dot).
Privacy: если сосед смотрит через плечо — тайная секция уже сверху и видна. Это осознанный компромисс: пользователь сам решает, когда открыть «Мои брони».
