North Star — Showcase profile (PRO)

Витрина как relationship anchor

Public profile — одна из 4 новых PRO-фич (plan_pro_f18). Это отдельная monetisation dimension: не «unlock feature», а «своя identity в WishBoard».

Cover gradient — мягкий premium signal

160px cover с multi-stop gradient (accent → warning). В current prod это обычно user-uploadable — здесь в mock показан default premium-gradient.

Profile identity block

  • 112px avatar в gradient-кольце, bordered in bg-color (создаёт «приподнятость» над cover)
  • Имя + PRO-бейдж — 22/800 title + pill-chip с accent-градиентом
  • Handle + дата — «@annasm · WishBoard с января 2025» — social signal долгосрочного отношения
  • Bio — 14px обычный текст. Личный голос (эмодзи, упоминание аллергии) вместо клинического описания.

Subscribe CTA

Primary-gradient «+ Подписаться на профиль» — главное действие для guest-viewer. Это entry в long-term relationship: ты получаешь все обновления её вишлистов.

Stats row

7 вишлистов / 142 желания / 34 подписчика — social proof. Помогает понять «эта учётка активная».

Размеры и предпочтения

  • Grid: одежда M · обувь 37 · кольцо 16.5 — practical данные для дарения
  • Brand-chips: Diptyque · Molskine · Aesop — subtle preference signal

Don't-gift banner сверху

Red-tinted banner «🚫 Что не дарить» — важнее чем preferences, показываем выше. Aligns с principle «state is unmistakable».

Pinned wishlists

Warning-tinted card с «ЗАКРЕПЛЁН» badge. Владелец профиля выбирает 1-3 закреплённых. «Скоро» chip для тех у кого дедлайн близко.

All wishlists list

Компактные rows с emoji + name + count. Легче сканировать чем полноразмерные карточки на 7 штук.
