North Star — Secret Santa

Sub-product внутри WishBoard

Тайный Санта — отдельное мини-приложение внутри WishBoard. Свои экраны, своя логика: жеребьёвка, alias'ы, exclusions, chat, polls.

Phone 1 — participant view

  • Santa-hero: deep-green / red gradient (зима/праздник, не бренд-purple). ❄️ декор, «🎅 Тайный Санта». Status-pill с animated green dot = «активная».
  • Alias card: «Ты в этой игре — 🦌 Храбрый Лось» — animal emoji 56px с shadow, name в prio-1 blue (отличается от accent-purple чтобы не путать с brand). Hint: «Никто не знает что это ты».
  • Assignment card (accent-purple tint): «🎁 Ты даришь — 🦊 Рыжему Лису». Бюджет в той же строке. CTA «📋 Посмотреть вишлист получателя».
  • Participants grid: 2-column grid с animal-emoji. Твой alias отмечен «ты», получатель отмечен «твой получатель». Остальные — просто «в игре» (anonymous).
  • Poll: inline bar-chart с lead-option выделен accent-border. 5 из 8 проголосовали — progress social.

Phone 2 — organizer view

  • Status tiles: «✅ жеребьёвка прошла» + «🎁 8 назначено». «Ты не видишь кто кому достался» — privacy guarantee.
  • Gift progress: 5 состояний с цветными dots (куплено / выбрано из вишлиста / сам выбирает / без вишлиста / не определился). Агрегированные counts — organizer видит прогресс, не кто-кому.
  • Timeline: 4 этапа (приглашения → жеребьёвка → выбор → встреча). Пройдённые — green-check, текущий — accent-ring, будущие — dashed.
  • Sticky footer: «📧 Отправить напоминание» (surface) + «Отменить кампанию» (danger-text, тихая).

Уникальный визуальный язык

  • Animal emoji + adjective вместо имён: 🦌 Храбрый Лось / 🦊 Рыжий Лис / 🐼 Мудрая Панда — legit anonymous-friendly
  • Бренд-gradient not used: Santa имеет свой green/red gradient. Это отдельный sub-product, не должен смешиваться с main app
  • Role-specific UI: participant видит assignment + alias, organizer видит aggregate progress + timeline
  • Privacy-first: «не видит кто кому» — ясно прописано и показано что organizer не получает доступа к draw

Что лучше current prod

  • Alias-card визуально отдельный и крупный — сейчас alias теряется
  • Assignment-card emphasizes «кто твой target» — главный интерес participant'а
  • Timeline для organizer'а — сейчас это растянуто по нескольким экранам
  • Aggregate gift-progress — организатор видит summary без нарушения privacy
  • Green/red seasonal gradient — Santa узнаётся с 1 взгляда, не путается с Paywall
