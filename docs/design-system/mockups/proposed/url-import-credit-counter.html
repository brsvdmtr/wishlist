URL-import — счётчик квоты + upsell

URL-импорт уходит от жёсткого 402. FREE-пользователь получает 5 импортов в месяц с явным счётчиком; PRO — безлимит. Когда бесплатная квота и платные кредиты исчерпаны — мягкий upsell с тремя выходами вместо стены.

Счётчик живёт под полем ввода ссылки на экране «Черновики». Тон меняется по остатку: ≥2 — accent, 1 — amber, 0 — danger + тап открывает шит. Месячный сброс — ленивый (бакет "YYYY-MM"), без шедулера.
