North Star — Group gift

Collaborative gifting — относительно уникальная фича

Не у каждого wishlist-app есть это. У WishBoard есть — и это одна из core-PRO фич (plan_pro_f3). Визуальный язык должен подчёркивать collaborative, не утилитарный.

Phone 1 — детали + прогресс

  • Hero wish preview: emoji-thumb 64px + price + rose-pink-tint gradient background. Показывает что собираем.
  • Progress card:
    • 78 000 ₽ крупно (24/800) в success-зелёном
    • «+23 000 ₽ за сегодня» — momentum indicator
    • Gradient-filled bar с glow
    • «60% — осталось 51 990 ₽» + дедлайн в footer
  • Participants: каждый с уникальным gradient-avatar. Роль (ОРГ бейдж). Amount внесён в success-зелёном. «Ожидаем» — pending-state с pending amount. «1 свободное место» — вход для приглашения.
  • Pinned organizer message: accent-tinted dashed-border — реквизиты карты, комментарий к переводу. Закреплено = visible inline.
  • Sticky CTA: «💳 Внести свою долю» — primary-gradient. Контекстно-явно, не generic «Continue».

Phone 2 — chat

  • Header: wish-thumb + название + «4 участника · 60%» — context sticky
  • Message bubbles: them (surface, tail slева) vs me (accent-purple, tail справа) — Telegram-style
  • System messages: dashed green border + italic — «Максим внёс 23 000 ₽ · 18%» — в потоке чата, не отдельной секции
  • Date separators: 11px muted center-aligned
  • Input: pill-field + round send-button в accent

Что лучше current prod

  • Progress-bar с glow эффектом + momentum signal «+X за сегодня» — сейчас просто number
  • Participants grid с pending-state и invite-slot — сейчас менее структурировано
  • Organizer-message закреплённая — чтобы реквизиты не терялись в чате
  • Contribution-events в чате — социальная прозрачность
  • Sticky CTA «Внести свою долю» — одно чёткое действие
